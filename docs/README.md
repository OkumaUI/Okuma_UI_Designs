# docs/

Project documentation and source files for the Okuma Parts Portal engagement.

## Contents

| File | Description |
|------|-------------|
| `okuma-parts-portal-design-spec.md` | Master design specification: page inventory, component patterns, color palette, typography, interaction rules, data model. Primary reference when building new pages or verifying design intent. |
| `stock-order-template.xlsx` | Excel template for dealer stock orders. Referenced in `create-order.html` as a downloadable template. |

## How to use
- **Before building a new page or component** — read `okuma-parts-portal-design-spec.md` first. It defines the shared design language (button styles, card patterns, color tokens, spacing scale).
- **When a dealer asks about bulk ordering** — `stock-order-template.xlsx` is the file they download from the dealer order flow.
